Returns all records from the right table and the matching records from the left table. The result is no records from the left side if there is not a match. From The Pieces SQL Collection. 
        SELECT column_name(s)
        FROM table1
        RIGHT JOIN table2
        ON table1.column_name = table2.column_name;
